# Claude Skills

このディレクトリには、Claude Skillsの実装が含まれています。各スキルはテンプレートを使用してインフラ操作を実行します。

## スキルカテゴリ

### provision/
新しいリソースを作成するスキル

例：
- VM/EC2インスタンスの作成
- ストレージアカウント/S3バケットの作成
- ネットワークリソースの作成

### configure/
既存リソースの設定を変更するスキル

例：
- リソースタグの更新
- アクセス権限の変更
- スケーリング設定の調整

### query/
リソース情報を取得するスキル

例：
- リソース一覧の取得
- リソース詳細情報の取得
- コスト情報の取得

### destroy/
リソースを削除するスキル

例：
- 単一リソースの削除
- リソースグループの削除
- クリーンアップ操作

### validate/
設定やテンプレートを検証するスキル

例：
- テンプレート構文チェック
- パラメータ検証
- セキュリティコンプライアンスチェック

## スキルの実装

各スキルは以下の責務を持ちます：

1. **入力解析**: ユーザーリクエストからパラメータを抽出
2. **テンプレート選択**: 適切なテンプレートを選択
3. **入力検証**: テンプレートスキーマに対して入力を検証
4. **コマンド生成**: CLIコマンドを生成
5. **実行**: 適切なエラーハンドリングでコマンドを実行
6. **結果返却**: 構造化された結果を返す

## スキルの構造

```
skill-name/
├── skill.yaml          # スキル定義
├── handler.py|sh       # 実行ハンドラ
├── README.md           # スキルドキュメント
└── tests/              # スキル固有のテスト
```

## Claude Skillsとの統合

スキルはClaude Skills frameworkを通じて公開されます：
- Claudeから構造化された入力を受け取る
- テンプレートベースの操作を実行
- 構造化された結果をClaudeに返す
- 同期・非同期操作の両方をサポート
